Quick Start

Get up and running in minutes

Install and Run 
# Install globally
npm install -g @r3e/neo-n3-mcp

# Run with testnet
neo-n3-mcp --network testnet

# Or run directly
npx @r3e/neo-n3-mcp
Docker Run 
# Run with Docker
docker run -p 3000:3000 \
  -e NEO_NETWORK=testnet \
  r3e/neo-n3-mcp:1.5.0

# With persistent storage
docker run -p 3000:3000 \
  -v $(pwd)/wallets:/app/wallets \
  r3e/neo-n3-mcp:1.5.0
Claude Desktop Configuration 
{
  "mcpServers": {
    "neo-n3": {
      "command": "npx",

Example Usage

Basic Operations 
# Get blockchain info
curl -X POST http://localhost:3000/mcp \
  -d '{"name": "get_blockchain_info"}'

# Create a wallet
curl -X POST http://localhost:3000/mcp \
  -d '{"name": "create_wallet", "arguments": {"password": "secure123"}}'

# Check balance
curl -X POST http://localhost:3000/mcp \
  -d '{"name": "get_balance", "arguments": {"address": "NZNos2WqTbu5oCgyfss9kUJgBXJqhuYAaj"}}'
