üöÄ NPM Installation
‚è±Ô∏è ~30 seconds 
# Install globally (recommended)
npm install -g @r3e/neo-n3-mcp

# Start with testnet
neo-n3-mcp --network testnet

# Or run directly without installation
npx @r3e/neo-n3-mcp

# Advanced: Custom configuration
neo-n3-mcp --network mainnet --port 3000 --log-level info

‚úÖ Requirements

  • Node.js 18+ (recommended: 20+)
  • NPM 8+ or Yarn 3+
  • Internet connection for blockchain data
üê≥ Docker Deployment
‚è±Ô∏è ~2 minutes 
# Quick start with Docker
docker run -p 3000:3000 \
  -e NEO_NETWORK=testnet \
  r3e/neo-n3-mcp:latest

# Production setup with persistent storage
docker run -d \
  --name neo-n3-mcp \
  --restart unless-stopped \
  -p 3000:3000 \
  -e NEO_NETWORK=mainnet \
  -e LOG_LEVEL=info \
  -v neo_wallets:/app/wallets \
  -v neo_logs:/app/logs \
  r3e/neo-n3-mcp:latest

üîß Configuration Options

  • NEO_NETWORK: mainnet, testnet
  • LOG_LEVEL: debug, info, warn, error
  • ENABLE_CONFIRMATION: true, false
